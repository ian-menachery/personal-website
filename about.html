Current Project: Mapping Asian Flight Routes

One of the most exciting projects I’m currently working on is a large-scale data visualization that maps commercial flight routes across Asia. The goal is to uncover geographic patterns in airline operations — such as which regions are heavily trafficked, how flight time correlates with distance, and how carriers differ in coverage.

I started by collecting and preprocessing a dataset containing thousands of flight records with detailed attributes like origin and destination coordinates, airline name, estimated distance, and flight time. After cleaning and formatting the data, I used Vega-Lite — a powerful grammar for interactive graphics — to create a layered map projection that plots each route as a curved line between airports. Airlines are color-coded, and users can hover over routes to view metadata like IATA codes, travel time, and carrier.

From a technical perspective, the project gave me the opportunity to deepen my understanding of:

  • Geospatial analysis — mapping coordinates onto Earth projections and tuning scale/center for legibility
  • Data wrangling — standardizing IATA codes, removing nulls, filtering by region, and handling multivalued airline fields
  • Declarative visualization — composing layered Vega-Lite specs for both the world map and flight arcs
  • User experience — choosing color palettes and interactivity tools that balance aesthetics with clarity

The project is still evolving — I’m currently working on adding filters to view routes by country, airline, or distance thresholds. I also plan to integrate airport metadata like city population or elevation to explore possible correlations. You can view the interactive visualization and the full source code on my GitHub.
